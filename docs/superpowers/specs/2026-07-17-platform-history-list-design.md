# 列表交互设计（2026-07-17）

## 已确认

- 平台分类栏单独一排：全部 / 小红书 / B站 / 抖音 / 快手 / X；字体加大；点一次切换。
- 去掉「未看 / 已看」筛选与「标已看」按钮。
- 点「打开」自动标记已看（链接旁 ☑️），写入 `watch_history`，累计 `open_count`。
- 「置顶」为手动置顶，永久直到取消。
- 非手动置顶按热度排序：`open_count / (1 + days_since_last_open / 7)`，再按收藏时间。
- 保留粘贴框 + 同步 Inbox，用于分享失败时的 fallback。

## 小红书文案

直接分享常只有 URL（系统未提供文案）；复制粘贴路径可带文案。联网补标题属后续增强。
